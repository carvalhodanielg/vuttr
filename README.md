## 	:toolbox: Getting Started

<!-- Prerequisites -->
### :bangbang: Prerequisites

This project uses npm as package manager

  
<!-- Run Locally -->
### :running: Run Locally

Clone the project

```bash
  git clone https://github.com/carvalhodanielg/countries.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
